# TlkEditor

Компактный редактор TLK-файлов (V1 и V3.0) для игр на движках Infinity и Aurora. Поддерживает сравнение двух файлов, продвинутый поиск и замену строк по фильтрам (флаги, звук, регистр, язык), систему закладок и обработку текста шаблонами.

## ОСНОВНЫЕ ВОЗМОЖНОСТИ

### Интерфейс и навигация:
- работа в окне фиксированного размера "960х540";
- навигация по строкам слайдером;
- отображение строк из двух файлов;
- открытие текущей строки в увеличенном окошке;
- вывод списка до 500 строк в отдельном окошке;
- создание списка закладок;
- навигация по найденным строкам через список закладок;
- навигация по изменениям строк;
- сохранение надписей интерфейса в файл .cfg рядом с EXE.

### Работа с форматами и файлами:
- поддержка TLK V1 (Baldur's Gate...) и TLK V3.0 (Neverwinter Nights...);
- поддержка внешнего файла для изменений строк и закладок;
- резервная копия файла при сохранении.

### Поиск, замена и фильтрация:
- поиск строк с отличиями в двух файлах;
- поиск или замена выполняется от текущей строки;
- поиск или замена строк с учётом или без учета регистра;
- точный поиск или замена строк;
- поиск или замена строк до 1 или для всех совпадений;
- исключающий поиск строк без замены текста;
- поиск строк среди закладок;
- поиск строк по флагу или по длине;
- поиск строк со звуковым ресурсом;
- поиск строк с русскими и/или английскими символами.

### Обработка текста:
- меню обработки текста строки шаблонами.
